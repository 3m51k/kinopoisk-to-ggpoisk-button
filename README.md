# GGpoisk Flag 🎬

Скрипт для [Tampermonkey](https://www.tampermonkey.net/), который добавляет стильную рыжую вертикальную закладку на страницы фильмов и сериалов **Кинопоиска**. При клике на закладку вы автоматически переходите на ту же страницу на сайте **GGpoisk** для бесплатного просмотра.

## Установка 📦

1. Установите расширение для запуска пользовательских скриптов:
   * [Tampermonkey](https://www.tampermonkey.net/) для Chrome, Firefox, Edge.

2. Установите сам скрипт, перейдя по этой ссылке:
   **[Установить скрипт GGpoisk Flag]([https://raw.githubusercontent.com/3m51k/kinopoisk-to-ggpoisk-button/main/ggpoisk-flag.user.js](https://github.com/3m51k/kinopoisk-to-ggpoisk-button/blob/main/kinopoisk-to-ggpoisk-button.js))**

3. Готово! Откройте любую страницу фильма на `kinopoisk.ru` — в левой части экрана появится рыжая закладка с треугольником. Нажмите на неё.

## Как это работает 🧠

*   Скрипт срабатывает только на страницах `kinopoisk.ru/film/...` и `kinopoisk.ru/series/...`.
*   Он создает элемент `<div>` с уникальным ID и применяет к нему стили, имитирующие бумажную закладку.
*   При клике JavaScript заменяет `kinopoisk` на `gg` в текущем URL и открывает новую вкладку.

## Внешний вид ✨

*   **Цвет:** Рыжий (`#ff6b35`), при наведении становится ярче (`#ff5722`).
*   **Позиция:** Вертикальная полоска, зафиксированная с левого края.
*   **Эффект:** При наведении мыши закладка плавно расширяется вправо.
